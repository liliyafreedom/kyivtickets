## 🚀 ІНСТРУКЦІЯ З НАЛАШТУВАННЯ (крок за кроком)

### КРОК 1 — Створи Telegram-бота (5 хвилин)

1. Відкрий Telegram, знайди **@BotFather**
2. Надішли йому: `/newbot`
3. Придумай назву (наприклад: `Kyiv Events`)
4. Придумай username (наприклад: `kyiv_events_may_bot`)
5. BotFather надішле тобі **TOKEN** — збережи його! Виглядає так:
   ```
   1234567890:AAHdqTcvCHhvWEoBPiG29_lSHSnpDvNGi-5
   ```

### КРОК 2 — Отримай свій Chat ID

1. Знайди в Telegram бота **@userinfobot**
2. Надішли йому `/start`
3. Він покаже твій **ID** (число, наприклад: `123456789`)
4. Збережи це число

**Або:** відкрий свого нового бота і натисни `/start`, потім перейди за посиланням:
```
https://api.telegram.org/bot<ТВІЙ_TOKEN>/getUpdates
```
Знайди `"chat":{"id":ЧИСЛО}` — це і є твій Chat ID.

---

### КРОК 3 — Завантаж код на GitHub

1. Зареєструйся на **github.com** (безкоштовно)
2. Натисни **"+"** → **"New repository"**
3. Назва: `kyiv-events-bot`
4. Постав галочку **"Add a README file"**
5. Натисни **"Create repository"**

Тепер завантаж файли:
1. Натисни **"Add file"** → **"Upload files"**
2. Завантаж `scraper.py` та `requirements.txt`
3. Натисни **"Commit changes"**

Для workflow файлу:
1. Натисни **"Add file"** → **"Create new file"**
2. В назві введи: `.github/workflows/events.yml`
3. Скопіюй вміст файлу `events.yml`
4. Натисни **"Commit changes"**

---

### КРОК 4 — Додай секрети (токен та chat id)

1. В репозиторії перейди: **Settings** → **Secrets and variables** → **Actions**
2. Натисни **"New repository secret"**
3. Додай перший секрет:
   - Name: `TELEGRAM_TOKEN`
   - Secret: вставте свій токен від BotFather
4. Натисни **"Add secret"**
5. Ще раз **"New repository secret"**:
   - Name: `TELEGRAM_CHAT_ID`
   - Secret: вставте свій Chat ID
6. Натисни **"Add secret"**

---

### КРОК 5 — Перший запуск

1. Перейди в **Actions** (вкладка у репозиторії)
2. Знайди **"Kyiv Events Monitor"**
3. Натисни **"Run workflow"** → **"Run workflow"**
4. Через ~2 хвилини перевір Telegram — мають прийти перші події!
